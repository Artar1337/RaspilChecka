<h1>Распил Чека</h1>

<h2>FAQ</h2>

<h3>Что это?</h3>
- Это - приложение для распила счета по чеку после посиделки.

<h3>Почему не гугл таблицы?</h3>
- Меня достали эти гугл таблицы, в которых мы с пацанами считаем деньги - неудобно и костыльно, так что я навайбкодил приложуху на Gemini Pro за +- 5 часов (это объясняет, почему я тут API ключ в открытом доступе положил, пж, не сносите мне базу данных, умоляю 🥺🥺🥺).

<h3>Могу ли я использовать приложение в личных целях?</h3>
- Да. Но с фото чеков шибко не увлекайтесь, хранилище тут всего 1 Gb.
